# Multiple-Disease-ML-App
Multiple Disease Prediction Web App using Python and Streamlit. Integrates separate ML models for Diabetes, Heart Disease, Parkinson’s Disease and Breast Cancer into a single interface, allowing users to enter clinical data and get instant risk predictions for each condition.

# Model Performance (Approx.)
-Breast Cancer: ~0.99 train accuracy, ~0.96 test accuracy
-Diabetes: ~0.78 train accuracy, ~0.77 test accuracy
-Heart Disease: ~0.85 train accuracy, ~0.82 test accuracy
-Parkinson’s Disease: ~0.88 train accuracy, ~0.87 test accuracy
